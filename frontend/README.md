# VolunteerHub Frontend

This is the new React-based frontend for VolunteerHub. It is designed to be modern, beautiful, and fully integrated with the backend API.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```

## Structure
- `src/pages/` — Main pages (Home, Opportunities, Profile, Admin, etc.)
- `src/components/` — Reusable UI components
- `src/api/` — API integration layer
- `src/styles/` — Global and component styles

## Features
- Responsive, modern UI
- Authentication (login/signup)
- Opportunities browsing, saving, and admin management
- AI-powered email and resume matching

For backend API, see the `../backend` folder.
